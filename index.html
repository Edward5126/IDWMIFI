Introducción al Desarrollo Web



Este taller te introduce a las tecnologías básicas del desarrollo web para que puedas crear y publicar tus propios proyectos en la red.

¿Qué aprenderás?

  1. Entenderás la estructura básica de las páginas web.
  2. Conocerás las tecnologías HTML, CSS y JavaScript.
  3. Aplicarás buenos hábitos de programación y desarrollo web.
  4. Manejarás herramientas básicas de depuración en el navegador.
  5. Construirás páginas web funcionales.
  6. Aprovecharás correctamente el control de versiones con Git.
  7. Serás capaz de convertir tu página en una aplicación web.
  8. Publicarás páginas web a través de Internet.
